你已通过 `/damn` 显式触发 **damn** 技能。

1. **加载技能正文**（按顺序，命中即停）：
   - **全局安装**：先 `Read` 用户主目录下的 `~/.claude/skills/damn/SKILL.md`（或 `$HOME/.claude/skills/damn/SKILL.md`）。
   - 若不存在，再检查当前仓库根目录的 `SKILL.md` 是否为本技能（含「智能技术调研」等标识）。
   - 若仍无，再查项目内 `.claude/skills/damn/SKILL.md`。
   - 读到正文后，严格按其阶段、模式与输出格式执行。
2. **用户本次输入**（`/damn` 后面的全部内容会替换下方占位符；若为空则不要编造主题）：

$ARGUMENTS

3. **空参数**：若上一步在去掉首尾空白后为空，用一句话请用户说明要评估的技术、方案或决策问题，然后从阶段一再开始。
4. **非空**：从 `SKILL.md` 中的**阶段一：问题理解与复杂度评估**开始，按路由选择快速 / 标准 / 深度模式，完成后续阶段直至最终报告（或低置信度时的暂停说明）。
